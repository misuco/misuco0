# Misuco HTML5 Port

## Overview
Misuco is a port of the original Misuco application rewritten in HTML5 and JavaScript. This project aims to provide a modern web-based version of the application that can run in any modern web browser.

## Features
- Run directly in the browser without any additional plugins.
- Full functionality of the original application.
- Responsive design to ensure compatibility across various devices.

## Getting Started
To run the Misuco HTML5 port locally, follow these steps:

1. **Clone the repository**:
   ```bash
   git clone https://github.com/misuco/misuco0.git
   cd misuco0
   ```

2. **Check out the HTML5 port branch**:
   ```bash
   git checkout html5-port
   ```

3. **Open the index.html file** in your preferred web browser.
   ```bash
   open index.html  # On macOS
   start index.html # On Windows
   xdg-open index.html # On Linux
   ```

4. **Enjoy using Misuco in your browser!**

## Direct Deployment
For deploying the project, you can host the files on any static file server or platform that supports HTML5 applications.
